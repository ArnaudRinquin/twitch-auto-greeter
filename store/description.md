## Name
Twitch Auto-Greeter

## Short Description (132 characters max)
Automatically send greetings in Twitch chat to be counted as an active viewer. Language-aware, customizable, privacy-focused.

## Detailed Description

Twitch recently changed how it counts spectators - only viewers who participate in chat are counted as active. Twitch Auto-Greeter ensures you're always counted by automatically sending a greeting message when you join a stream.

### How It Works

The extension uses a smart cascading message selection:

1. **Streamer-specific messages** are prioritized - if you've configured messages for a specific streamer, those are used exclusively
2. **Language-specific messages** are next - the extension detects stream language tags and sends appropriate greetings
3. **Global messages** as fallback - language-agnostic messages work on any stream
